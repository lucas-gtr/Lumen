Detailed Description

Class to manage the settings for rendering, including resolution, clipping planes, and other related parameters.

This class provides getter and setter methods for each rendering setting, with bounds checking to ensure valid values are used. The class supports image resolution, number of color channels, near and far clipping planes, maximum bounces for ray tracing, samples per pixel, and background color.
