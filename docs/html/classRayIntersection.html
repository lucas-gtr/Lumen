Detailed Description

Class responsible for ray-triangle and ray-mesh intersection calculations.

This class provides methods to calculate the intersection of rays with triangles and meshes. The main functions are getTriangleIntersection() for ray-triangle intersections and getMeshIntersection() for ray-mesh intersections.
