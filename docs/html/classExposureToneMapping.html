void ExposureToneMapping::apply ( double &  value) const
inlineoverridevirtual

Applies the exposure tone mapping algorithm to a given value.

Parameters
valueThe value to be tone-mapped.

This method modifies the input value in place using the exposure tone mapping formula. The formula is defined as: \( \text{output} = 1 - e^{-\text{input} \cdot \text{exposure}} \)

This formula ensures that the output value is clamped between 0 and 1.
