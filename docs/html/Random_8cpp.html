◆ randomPointInUnitDisk()

Eigen::Vector2d randomPointInUnitDisk ( )

Generates a random point inside the unit disk.

This function generates a random 2D point where both coordinates lie within the unit disk, i.e., the point satisfies the condition: \( x^2 + y^2 \leq 1 \).

The point is generated using a random angle and radius within the unit disk, ensuring uniform distribution within the disk.

Returns
A 2D vector (Eigen::Vector2d) representing the random point in the unit disk.
