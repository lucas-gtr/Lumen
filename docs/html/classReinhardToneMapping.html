void ReinhardToneMapping::apply ( double &  value) const
inlineoverridevirtual

Applies the Reinhard tone mapping algorithm to a given value.

Parameters
valueThe value to be tone-mapped.

This method modifies the input value in place using the Reinhard tone mapping formula. The formula is defined as: \( \text{output} = \frac{\text{input}}{1 + \text{input}} \) This formula ensures that the output value is clamped between 0 and 1.

Implements ToneMappingStrategy.
